# ides/ — multi-assistant exports

> _Generated by `tools/export_ides.py` from the `plugins/` SKILLs — do not edit by hand. The canonical source is each domain's `SKILL.md`._

These directories mirror the `plugins/` SKILLs into formats other AI coding assistants read, so the same analog/mixed-signal + RF domain knowledge is available outside Claude Code. The **source of truth is `plugins/<domain>/skills/<skill>/SKILL.md`**; regenerate after changing a SKILL with:

```bash
python3 tools/export_ides.py
```
